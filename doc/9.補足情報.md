## アセット

- `/torifuda` フォルダ内に、取り札の画像ファイルが PNG 形式で入っている
    - `tori_ura.png` は裏向きになった取り札の画像
- `fudalist.js` に、100首の歌の情報が入っている
    - `no`：歌番号
    - `kimariji`：決まり字
    - `normal`：取り札の正位置（自陣）での向きの画像ファイルへのパス
    - `reverse`：取り札の逆位置（相手陣）での向きの画像ファイルへのパス
    - `order`：定位置編集画面の未配置プールの並び順、および確認モード「決まり字順」のソートキー

## Web アプリ（リポジトリ構成）

- ルートに Vite + React + TypeScript プロジェクト（`npm run dev`）
- `src/data/fudalist.ts` … 歌データ（`npm run generate:fudalist` で `fudalist.js` から再生成）
- `public/torifuda` … `torifuda/` へのシンボリックリンク（静的配信）
