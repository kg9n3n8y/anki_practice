## トップ画面

- 画面上部にアプリ名 **「かるた暗記練」** を表示
- 暗記練習スタート画面や定位置編集画面へ遷移する
- **最近の成績**はやや小さめの文字サイズで表示
- 項目
    - **最近の成績**（直近5回）
        - 日時は `2026/6/4 23:25` のように**秒なし**
        - 各回：枚数設定、暗記モード、制限時間、正答数（0.5 点を含みうる）、出題数、正答率、確認モードにかかった時間 など
    - **「暗記練習を始める」** → 暗記練習スタート画面
    - **現在の定位置の保存状態**（保存済み / 未保存）
    - **「定位置を編集する」** → 定位置編集画面（`/position`）
    - **「定位置の取りこみ」** → ダイアログでコンパクトテキストを貼り付けてインポート
- 画面最下部（小さめ）：**URLをコピー**（`https://kg9n3n8y.github.io/anki_practice/` をクリップボードへ）、**作者: つばさ先輩**（[作者サイト](https://sites.google.com/view/hyakunin-issyu-oboekata/) へのリンク・別タブ）
- **初回のみ**：ホーム画面に追加の案内ダイアログ（すでに追加済み・「了解」後は LocalStorage で非表示。iOS/Android で文言を切り替え）

## 暗記練習スタート画面

- 暗記練習のモードや制限時間を設定する
- 各設定項目は前回の選択状態を LocalStorage 等で保持する
- 設定項目は **2列グリッド**（狭い画面では1列）で、次の3行に配置する
    - 1行目：**モード**｜**自陣の配置**（プルダウン：定位置を適用／ランダム）
    - 2行目：**取り札の枚数**｜**制限時間**
    - 3行目：**確認モードの空札の枚数**｜**確認モードの出題順**
- **モード選択**（プルダウン）
    - 「相手陣のみ」「自陣のみ」「両方」（デフォルト：**相手陣のみ**）
- **取り札の枚数**（プルダウン、デフォルト：**25枚**／両方モード初回切替時は50枚）
    - 「相手陣のみ」「自陣のみ」：5〜25枚、5枚単位（その陣のみに配置。他陣は非表示）
    - 「両方」：10〜50枚、10枚単位（**合計枚数**。常に偶数のため自陣・相手陣は各 `N/2` 枚。例：50 → 各25）
- **制限時間**（プルダウン）：1〜15分、1分単位（デフォルト：**5分**）
- **自陣の配置**（プルダウン）
    - 「定位置を適用」「ランダム」（デフォルト：ランダム。未保存・相手陣のみのときは選択不可でランダム固定）
- **確認モードの空札の枚数**（プルダウン、デフォルト：**15枚**／両方モード初回切替時は30枚）
    - 確認モードで盤外から出題する枚数
    - 「相手陣のみ」「自陣のみ」：0、5〜25枚、5枚単位
    - 「両方」：0、10〜50枚、10枚単位
    - 出題総数・選定ルールの詳細は `2.要件定義.md` の確認モードを参照
- **確認モードの出題順**（プルダウン、デフォルト：決まり字順）
    - 決まり字順：出題札を `order` 昇順
    - ランダム順：出題札を完全ランダム順

## 暗記練習メイン画面

- ツールバーは **3列**（左：完了/空札、中央：残り時間または決まり字、右：中止）
- 暗記モードの残り時間表示は、確認モードの決まり字と同程度の大きさ

### 盤面表示

- **相手陣のみ**：相手陣にのみ札を配置し、自陣は非表示
- **自陣のみ**：自陣にのみ札を配置し、相手陣は非表示
- **両方**：合計枚数に応じて自陣・相手陣に配分して配置

### 初期配置

- 各陣の6エリアへ、外側詰めでランダム配置（1エリア最大7枚）
- 盤面表示：1段の幅 = 左7枚 + 隙間1枚 + 右7枚（計15枚分）。取り札サイズはエリアによらず統一
- 自陣で定位置 ON の場合：選出された登録歌は**必ず**指定エリアへ **保存した並び順**で配置（空エリアあり可）。定位置未登録の選出札のみランダム。選出外の定位置は無視

### 暗記時間

- 配置された取り札の位置を暗記する
- ツールバー：**左**に「完了」、**中央**に残り時間、**右**に「中止」（赤）
- 「完了」で残り時間があっても確認モードへ移行可能（確認ダイアログあり）

### 確認モード

- 盤上の札をすべて裏向き（`tori_ura.png`）にする。裏向き画像の読み込み・描画が完了してから1問目の決まり字を表示する（移行直後のちらつき防止）
- ツールバー：**左**に「空札」（緑・完了ボタンと同系。空札0枚のときは非表示）、**中央**に決まり字（ラベルなし・大きく表示）、**右**に「中止」（赤）
- 盤上の全札が表向きになったら：**左**の「空札」を非表示、**右**の「中止」を「結果」（緑・「空札」と同系）に変更。ユーザーが「結果」を押すまで盤面を表示したまま（スクショ用）
- 決まり字を提示。ユーザーは該当札の位置をタップ、盤に無い場合は「空札」
- **出題**：盤使用枚数 ＋ 確認モードの空札の枚数。選定は「使用中の札 + 盤外からランダム」。順序はスタート画面の設定に従う
    - **正誤表示**
        - ツールバーの決まり字の文字色：正解＝赤、不正解＝青、近接正解（0.5）＝緑
        - 盤上の歌のみ、表にした札の画像上に **○／×／△** を各文字色で **20% 不透明** で重ねる（確認モード終了まで残す。盤外の出題では札上マークなし）。正解の ○ は文字ではなく CSS の円で描画（二重線を避ける）
        - 正解：タップした札が表向き。近接正解・不正解（「空札」含む）：正解位置の札が表向き。盤外正解は札は裏のまま
        - 近接正解は同一エリアの左右隣タップで正答 **0.5** 加算
    - 表札画像は確認モード開始時に先読みし、裏→表の切り替えを即時に近づける
- **終了**：盤上の全札が表向きになった時点（出題リストの全問消化が条件ではない）。「結果」ボタン押下で結果画面へ
- 解答から次の出題まで約 0.7 秒
- **結果画面**
    - 正答率：正答数 / 出題した決まり字の数（近接正解があると正答数は `12.5` のように小数表示）
    - 確認モードにかかった時間
    - 成績を LocalStorage に保存（直近5回まで保持）

## 定位置編集画面

- パス: `/position`（旧 `/teigi` はリダイレクト）
- 自陣の定位置を作成・編集・保存・リセットする（相手陣の編集はしない）
- 6エリアの配置図（エリア名ラベルは表示しない）：

| （左上段） | （右上段） |
| ------ | ------ |
| （左中段） | （右中段） |
| （左下段） | （右下段） |

- 右側エリア（右上・右中・右下）は決まり字を**右揃え**
- 各エリア内で D&D により**並び順**を変更可能
- 下部の未配置プール：**縦書き**・`order` 昇順。プール内は並べ替え不可
- 決まり字表示：枠なしの縦書きテキスト。5文字は `AB・E`、6文字は `AB・F` 形式（例：`よのなかよ` → `よの・よ`）
- **保存** / **テキストをコピー**（クリップボード・コンパクト形式）/ **リセット** / トップへ戻る
- **保存**：LocalStorage（v2 JSON）。次回表示時に復元
- **テキストをコピー**：1行1エリア形式（例 `leftUpper:3,12`）。トップの取りこみと共有
- プールからエリアへ、およびエリア間の D&D では、ドロップ位置に薄いプレビューを表示
- プールからエリアへ D&D する際、既存の決まり字の**左側に差し込むか右端に置くか**をドロップ位置で指定可能
