## トップ画面

- アプリ名・説明（仮文言は `0.概要.md` を参照）を表示する
- 暗記練習スタート画面や定位置編集画面へ遷移する
- 項目
    - **最近の成績**（直近5回）
        - 各回：枚数設定、暗記モード、制限時間、正答数、出題数、正答率、確認モードにかかった時間 など
    - **「暗記練習を始める」** → 暗記練習スタート画面
    - **現在の定位置の保存状態**（保存済み / 未保存）
    - **「定位置を編集する」** → 定位置編集画面
    - **「定位置の取りこみ」** → ダイアログで定位置 JSON をインポート（エクスポートはこの画面では行わない）

## 暗記練習スタート画面

- 暗記練習のモードや制限時間を設定する
- 各設定項目は前回の選択状態を LocalStorage 等で保持する
- 設定項目
    - **モード選択**（プルダウン）
        - 「相手陣のみ」「自陣のみ」「両方」（デフォルト：両方）
    - **取り札の枚数**（プルダウン）
        - 「相手陣のみ」「自陣のみ」：5〜25枚、5枚単位（その陣のみに配置。他陣は非表示）
        - 「両方」：10〜50枚、10枚単位（**合計枚数**。常に偶数のため自陣・相手陣は各 `N/2` 枚。例：50 → 各25）
    - **制限時間**（プルダウン）：1〜15分、1分単位（デフォルト15分）
    - **自陣の定位置**
        - 定位置を適用するか（保存済みならデフォルト ON、未保存なら OFF のみ選択可）
    - **確認モードの出題順**（プルダウン、デフォルト：決まり字順）
        - 決まり字順：出題札を `order` 昇順
        - ランダム順：出題札を完全ランダム順
        - 出題数・札の選定ルールは `2.要件定義.md` の確認モードを参照

## 暗記練習メイン画面

### 盤面表示

- **相手陣のみ**：相手陣にのみ札を配置し、自陣は非表示
- **自陣のみ**：自陣にのみ札を配置し、相手陣は非表示
- **両方**：合計枚数に応じて自陣・相手陣に配分して配置

### 初期配置

- 各陣の6エリアへ、外側詰めでランダム配置（1エリア最大6枚）
- 盤面表示：1段の幅 = 左6枚 + 隙間1枚 + 右6枚（計13枚分）。取り札サイズはエリアによらず統一
- 自陣で定位置 ON の場合：登録歌は指定エリアへ。選出に含まれない定位置歌は無視。それ以外はランダム

### 暗記時間

- 配置された取り札の位置を暗記する
- 残り時間タイマーを表示
- 「完了」で残り時間があっても確認モードへ移行可能

### 確認モード

- 盤上の札をすべて裏向き（`tori_ura.png`）にする
- 決まり字を提示。ユーザーは該当札の位置をタップ、盤に無い場合は「なし」
- **出題**：盤使用枚数の2倍。選定は「使用中の札 + 残り札からランダム」。順序はスタート画面の設定に従う
    - **正誤表示**
        - 正解：⭕️、該当札が表向き（盤外正解の場合は表向きにしない）。ブラウザビープ等
        - 不正解：❌ のみ表示。該当札が表向きになる（盤外の歌へ誤って札をタップした場合は表向きにしない）。「なし」誤答時は正解位置の札を表向きにする
- **終了**：盤上の全札が表向きになった時点
- **結果画面**
    - 正答率：正答数 / 出題した決まり字の数
    - 確認モードにかかった時間
    - 成績を LocalStorage に保存（直近5回まで保持）

## 定位置編集画面

- 自陣の定位置を作成・編集・保存・JSON 出力する（相手陣の編集はしない）
- 6エリアの配置図：

| 自陣 左上段 | 自陣 右上段 |
| ------ | ------ |
| 自陣 左中段 | 自陣 右中段 |
| 自陣 左下段 | 自陣 右下段 |

- 下部に **縦書き** の決まり字リスト（`order` 順）。リストから各エリアへ D&D で「歌 → エリア」を登録
- **保存**：LocalStorage。次回表示時に復元
- **出力**：JSON エクスポート（スキーマは `4.技術・データ仕様.md`）
- インポートはトップ画面の「取りこみ」から行う
