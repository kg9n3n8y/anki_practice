## アプリ名（仮）

**かるた配置暗記**

## 説明（仮）

競技かるたの暗記時間向けに、取り札の配置を覚え、決まり字から位置を思い出す練習ができる Web アプリです。大会形式に近い盤面表示と、自陣の定位置を使った練習に対応します。

## 目的

競技かるたの暗記時間における配置記憶を効率化し、短時間で正確に札の位置を把握・想起できる能力を養う。

## 背景

実際の大会では15分間で50枚の札を覚える必要があるが、一人で練習する際に効率的な暗記確認ツールが少ない。

## ターゲットユーザー

- 競技かるた初心者：配置の基本を覚えたい
- 有段者・選手：より早く、正確な配置記憶と決まり字の連動を鍛えたい

## スコープ（Phase 1）

- トップ画面、暗記練習スタート画面、暗記練習メイン画面、定位置編集画面
- 暗記モード（相手陣のみ / 自陣のみ / 両方）と確認モード
- 自陣定位置の保存・JSON エクスポート（編集画面）・インポート（トップ画面）
- 成績の LocalStorage 保存（直近5回）
- GitHub Pages 向け静的配信、PWA（Service Worker + manifest）
- モバイルの陣ズームは Phase 2 以降
