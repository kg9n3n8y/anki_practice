## アプリ名

**かるた暗記練**

## 説明

競技かるたの暗記時間向けに、取り札の配置を覚え、決まり字から位置を思い出す練習ができる Web アプリです。大会形式に近い盤面表示と、自陣の定位置（エリア内の並び順を含む）を使った練習に対応します。

## 目的

競技かるたの暗記時間における配置記憶を効率化し、短時間で正確に札の位置を把握・想起できる能力を養う。

## 背景

実際の大会では15分間で50枚の札を覚える必要があるが、一人で練習する際に効率的な暗記確認ツールが少ない。

## ターゲットユーザー

- 競技かるた初心者：配置の基本を覚えたい
- 有段者・選手：より早く、正確な配置記憶と決まり字の連動を鍛えたい

## スコープ（Phase 1・実装済み）

### 画面

| パス | 画面 |
| --- | --- |
| `/` | トップ（成績・定位置取りこみ・フッター） |
| `/practice/start` | 暗記練習スタート |
| `/practice/main` | 暗記練習メイン（暗記・確認・結果） |
| `/position` | 定位置編集 |
| `/teigi` | `/position` へリダイレクト（旧URL互換） |

### 機能

- 暗記モード（相手陣のみ / 自陣のみ / 両方）と確認モード
- 自陣定位置の編集（D&D・エリア内/間の並べ替え・保存）
- 定位置の共有：コンパクトテキストのコピー（編集画面）・取りこみ（トップ）
- 確認モード：空札枚数の設定、近接正解（0.5点）、決まり字色・札上マークによる正誤表示
- 成績の LocalStorage 保存（直近5回）
- GitHub Pages 配信、PWA（Service Worker・manifest・アイコン）
- 効果音（Web Audio API・正解/近接正解/不正解）
